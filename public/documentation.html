Movie API Documentation

This API will provide interaction with a movie database and a user database.

User Database

Method URL endpoint Description
POST /users adds a new user

Request

Format: JSON
JSON parameters:
name (required) - string, unique username
password (required) - string
email (required) - string
birthday - Date
Example request body:
{
  "name": "Alice",
  "password": "mypassword",
  "email": "testemail@email.com"
}

Response

JSON of user data
Format: JSON
Example response: 
{
  "name": "Alice",
  "email": "testemail@email.com",
  "favorites": [],
  "_id": "646f50fb47f96ed7d4af88f2",
  "__v": 0
}
PUT /users/[ID] updates user data by user ID

Request

URL parameter:
ID (required) - number, user ID
Format: JSON
JSON parameters:
name - string, unique username
password - string
email - string
birthday - Date
Example request body:
{
  "email": "newtestemail@email.com"
}

Response

the updated user data
Format: JSON
Example response: 
{
  "_id": "646f50fb47f96ed7d4af88f2",
  "name": "Alice",
  "email": "newtestemail@email.com",
  "favorites": [],
  "__v": 0
}

GET /users/[ID] gets user data

Request

URL parameters:
ID (required) - number, user ID
Example request:
GET /users/64687fdaf5a9c387994faffd

Response

Format: JSON
Example response: 

{
  "_id": "646f50fb47f96ed7d4af88f2",
  "name": "Alice",
  "email": "testemail@email.com",
  "favorites": [],

POST /users/[userID]/movies/[movieID] adds a movie to a user's list of favorites

Request

URL parameters:
userID (required) - number, user ID
movieID (required) - number, movie ID
Example request:
POST /users/64687fdaf5a9c387994faffd/movies/6467dcfaaf8963a497ac08b5

Response

updated user's favorites list
Format: array
Example response:
[ "6467dcfaaf8963a497ac08b9", "6467dcfaaf8963a497ac08ba", "6467dcfaaf8963a497ac08b5" ]
DELETE /users/[userID]/movies/[movieID] removes a movie from a user's list of favorites

Request

URL parameters:
userID (required) - number, user ID
movieID (required) - number, movie ID
Example request:
DELETE /users/64687fdaf5a9c387994faffd/movies/6467dcfaaf8963a497ac08b5

Response

updated user's favorites list
Format: array
Example response:
[ "6467dcfaaf8963a497ac08b9", "6467dcfaaf8963a497ac08ba" ]
DELETE /users/[ID] removes a user by ID

Request

URL parameters:
ID (required) - numer, user ID
Example request:
DELETE /users/646f50fb47f96ed7d4af88f2

Response

a message confirming the deletion
Format: text
