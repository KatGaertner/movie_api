Movie API Documentation

This API will provide interaction with a movie database and a user database.

API Reference

Movie Database

Method URL endpoint Description
GET /movies returns a list of all movies

Request

Example request:
GET /movies

Response

Format: JSON
Response fields:
name (default) - string
directors (default) - array
genres (default) - array
Example response: 
[
  {
    "title": "Everything Everywhere All at Once",
    "directors": [ "Daniel Kwan", "Daniel Scheinert" ],
    "genres": ["Action", "Adventure", "Comedy"]
  },
  {
    "title": "All Quiet on the Western Front",
    "directors": [ "Edward Berger" ],
    "genres": [ "Action", "Drama", "War" ]
  },
  ...
]
GET /movies/[title] returns data about a single movie by title

Request

URL parameter:
title (required) - string, full title of the movie
Example request:
GET /movies/Everything%20Everywhere%20All%20at%20Once

Response

Format: JSON
Response fields:
name (default) - string
directors (default) - array
genres (default) - array
Example response: 
{
  "title": "Everything Everywhere All at Once",
  "directors": [ "Daniel Kwan", "Daniel Scheinert" ],
  "genres": [ "Action", "Adventure", "Comedy" ]
}
GET /genres/[name] returns data about a genre by name

Request

URL parameter:
genre (required) - string
Example request:
GET /genres/Action

Response

Format: JSON
GET /directors/[name] returns data about a director by name

Request

URL parameter:
director (required) - string
Example request:
GET /directors/Daniel%20Kwan

Response

Format: JSON

User Database

Method URL endpoint Description
POST /users adds a new user

Request

Format: JSON
JSON parameters:
name (required) - string, username
Example request body:
{
  "name": "Claire"
}

Response

JSON of user data
Format: JSON
Example response: 
{
  "name": "Claire",
  "id": "a9faacde-b747-42b6-a50f-19160682d073",
  "favorites": []
}
PUT /users/[id] updates user data by id

Request

URL parameter:
id (required) - string, user ID
Format: JSON
JSON parameters:
the to-be-updated fields
Example request body:

Response

the updated user data
Format: JSON
Example response:
POST /users/[id]/movies/[name] adds a movie to a user's list of favorites

Request

URL parameters:
id (required) - string, user ID
title (required) - string, movie title
Example request:
POST /users/1234/movies/Elivs

Response

updated user's favorites list
Format: array
Example response:
[ "The Banshees of Inisherin", "Elvis" ]
DELETE /users/[id]/movies/[name] removes a movie from a user's list of favorites

Request

URL parameters:
id (required) - string, user ID
title (required) - string, movie title
Example request:
DELETE /users/1234/movies/Elivs

Response

updated user's favorites list
Format: array
Example response:
[ "The Banshees of Inisherin" ]
DELETE /users/[id] removes a user by id

Request

URL parameters:
id (required) - string, user ID
Example request:
DELETE /users/1234

Response

a message confirming the deletion
Format: text
